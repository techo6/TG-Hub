# Total Gaming Esports Hub 🎮

A fully responsive, animated esports website built for the **Total Gaming** brand.
Deployed live on **Netlify**.

## 🌐 Live Demo
[View on Netlify](#your-netlify-link-here)

## 📌 Features
- 🔐 Login / Register system with localStorage auth state
- 🏆 Player Roster with flip cards
- 📅 Match Schedule with reminder button
- 📊 Tournament Standings table
- 🛒 Merch Shop with spotlight effect
- 🎥 Media & Video Gallery
- 🖼️ Infinite image gallery slider
- 📬 Support Desk (auth-protected ticket form)
- 🌀 Custom animated cursor (GSAP)
- 📱 Fully mobile-responsive with hamburger menu

## 🛠️ Tech Stack
- HTML5, CSS3, JavaScript (Vanilla)
- GSAP (animations & scroll triggers)
- Google Fonts (Orbitron, Rajdhani)
- Netlify (deployment)

## 📁 File Structure
├── index.html       → Main landing page
├── login.html       → Authentication page
├── support.html     → Support desk (login required)
├── style.css        → Main stylesheet
├── mobile-fix.css   → Mobile responsive overrides
├── script.js        → All JS logic & animations
└── assets/          → Images & media
